# working-with-Taiga-APIs

## Description
This is a React application. Suggest to use Visual Studio Code to run.

Youtube Demo: https://youtu.be/lQVoW6vCMu4

## How to Run

1. navigation to folder where these files are stored. (Make sure you're in 'working-with-taiga-api' folder)
2. type: npm install
3. type: npm start, or: yarn start
4. A web page should open, enter your Taiga.io username and password, then click 'Log in'
5. Lots of text will show in the web page, click a project you want, then a sprint if there is any
